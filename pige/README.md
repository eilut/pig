# منصة تسجيل المستفيدين للبرامج التدريبية

هذا المستودع مخصص لتوثيق وتحضير تنفيذ منصة إلكترونية لإدارة تسجيل الطلاب والطالبات في البرامج التدريبية بمحافظة المحويت.

## الهدف

بناء نظام يدير:

- المديريات المعتمدة
- البرامج التدريبية
- المقاعد حسب الجنس
- منع التسجيل المكرر
- إغلاق البرامج الممتلئة
- إدارة بيانات المستفيدين
- التقارير والتصدير
- لوحة تحكم إدارية باللغة العربية وبتخطيط RTL
