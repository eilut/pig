# الخطة التنفيذية

## الهدف من هذه الخطة

تحويل متطلبات النظام إلى مراحل تنفيذية واضحة قابلة للتحديث مع التقدم في العمل.

## المرحلة 1: التأسيس والتحليل

### الأهداف

- تثبيت نطاق النظام النهائي.
- اعتماد الجداول الأساسية.
- اعتماد تدفق التسجيل.
- تحديد الصلاحيات ونقاط الحماية.

### المخرجات

- وثيقة تحليل.
- خطة تنفيذ.
- سجل تنفيذ مبدئي.

### الحالة

- مكتملة مبدئيًا.

## المرحلة 2: تصميم قاعدة البيانات

### الأهداف

- تصميم الجداول والعلاقات.
- إضافة القيود لمنع التكرار.
- تحديد الفهارس الأساسية للأداء.

### المخرجات

- مخطط قاعدة البيانات.
- سكربت إنشاء الجداول.

### الحالة

- قيد التنفيذ.

### ما تم إنجازه حتى الآن

- إعداد سكربت أولي لقاعدة البيانات في backend/sql/schema.sql.
- تثبيت القيود الأساسية لمنع تكرار رقم البطاقة ورقم الهاتف.

## المرحلة 3: بناء الخلفية

### الأهداف

- إنشاء API للتسجيل والإدارة.
- بناء المصادقة ولوحة الإدارة.
- تنفيذ منطق التحقق من المقاعد والتكرار.
- بناء خدمات التقارير والتصدير.

### المخرجات

- خوادم API.
- طبقة صلاحيات.
- طبقة تحقق.

### الحالة

- قيد التنفيذ فعليًا داخل Docker.

### ما تم إنجازه حتى الآن

- إنشاء نواة منطق التسجيل في backend/src/registration-rules.js.
- إضافة اختبارات تلقائية لقواعد المقاعد وحالة البرنامج ومنع التكرار.

## المرحلة 4: بناء واجهة التسجيل العامة

### الأهداف

- إنشاء صفحة التسجيل.
- ربط المديريات بالبرامج.
- إخفاء البرامج غير المتاحة.
- عرض الرسائل المناسبة للمستخدم.

### المخرجات

- واجهة عربية RTL.
- نموذج تسجيل تفاعلي.

### الحالة

- قيد التنفيذ فعليًا داخل Docker.

## المرحلة 5: بناء لوحة الإدارة

### الأهداف

- إدارة البرامج والمديريات والمقاعد.
- إدارة بيانات الطلاب.
- عرض التقارير.
- تنفيذ التصدير.

### المخرجات

- لوحة تحكم إدارية.
- جداول بحث وتصفية.
- شاشات تقارير.

### الحالة

- لم تبدأ.

## المرحلة 6: الأمان والاختبارات

### الأهداف

- اختبار منع التكرار.
- اختبار إغلاق البرامج الممتلئة.
- اختبار الصلاحيات.
- اختبار الاستجابة على الهاتف والكمبيوتر.

### المخرجات

- تقرير اختبار.
- إصلاحات نهائية.

### الحالة

- بدأت.

## المرحلة 7: النشر والتشغيل

### الأهداف

- تجهيز بيئة النشر على VPS.
- ضبط قاعدة البيانات والنسخ الاحتياطي.
- تشغيل النظام ومراقبته.

### المخرجات

- نسخة إنتاجية.
- دليل تشغيل مختصر.

### الحالة

- بدأت على مستوى البيئة المحلية بالحاويات.
